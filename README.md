# Snake Game

Welcome to the Snake Game! This is an advanced version of the classic Snake game, featuring a range of customizable options and additional gameplay elements. This project is built using HTML, CSS, and JavaScript.

## Features

- **Customizable Snake Color**: Change the color of the snake to your preference.
- **Toggling Obstacles**: Enable or disable obstacles in the game.
- **Leaderboard**: Track and display high scores.
- **Special Food**: Gain extra points by collecting special food.
- **Power-Ups**: Collect power-ups to boost your score.
- **Difficulty Levels**: Game difficulty increases as you score more points.
- **Score History**: View a history of your scores.
- **Game Speed Adjustment**: Adjust the game speed to match your skill level.
- **Pause and Resume**: Pause and resume the game at any time.

## Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/pragunsrv/snake-game.git
   cd snake-game
   ```

2. **Open the Project**

   Open the `index.html` file in your web browser to start playing the game. You can do this by double-clicking the file or using a local web server.

## Usage

1. **Controls**
   - **Arrow Keys**: Move the snake up, down, left, or right.
   - **Enter**: Restart the game if you lose.
   - **Space**: Pause or resume the game.
   - **+ / -**: Adjust the game speed.

2. **Game Settings**
   - **Game Speed**: Adjust the speed of the game using the slider in the settings.
   - **Toggle Obstacles**: Check or uncheck the box to enable or disable obstacles.
   - **Snake Color**: Choose your preferred snake color using the color picker.

3. **Viewing Scores**
   - **High Score**: The highest score achieved during the game.
   - **Current Score**: The score of the current game session.
   - **Leaderboard**: View a list of top scores.

4. **Game Over**
   - When the game is over, the screen will display a "Game Over" message and the option to press Enter to restart.

## Project Structure

- **index.html**: The main HTML file that sets up the game canvas and interface.
- **styles.css**: The CSS file that styles the game and settings interface.
- **script.js**: The JavaScript file that contains the game logic and functionality.
